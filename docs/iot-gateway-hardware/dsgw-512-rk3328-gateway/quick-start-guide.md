## **1. Product Introduction**

This Quick Start Guide explains the basics:</br />
- how to connect and set up your target on the network</br />
- how to install the SDK</br />
- how to modify and build the firmware images</br />
The Linux Software Developer’s Kit (SDK) is an embedded hardware and software suite that enables Linux developers to create applications on Dusun’s DSGW-512 gateway for Yocto.</br />


## 2. Gateway Information
This section describes the gateway's basic resource infomation and interfaces.

### 2.1 Basic information

- Processor: RK3328 (ARM64)
- Supply: DC-12V/2A
- RAM: 2G
- EMMC: 8G
- Ethernet WAN: WAN RJ45/10M/100M
- WIFI: RTL8821CS
- Zigbee: EFR32MG1B232
- LED: 6leds{led4g/ledwifi/ledzgw/led1/led2/ledaux}
- RTC: PCF85063ATT
- BTN: 1
- DI: 1
- AO: 1
- NTC: 1
- relay: 8
- LTE: EC25
- USB: 1
- OTG: 1
- DI2: 1
- r485_1: 1
- mbus: 1

## 4. SDK Download And Compile
This section describes how to download the sdk and compile it.

### 4.1 SDK Envirment Prepare
Compilation environment: Ubuntu22.04
Openwrt's compilation tool is automatically generated by SDK built, no additional
installation is required

- make sure your machine have enough ram (2G+)

### 4.2 SDK Download
Get the source code from Dusun FTP server uncompress it under your work directory.
For example:

```shell
mkdir -p ~/workdir
tar zxvf DSGW-511-DB-LUCI-SDK_20260430.tar.gz -C /workdir/
cd ~/workdir/
```

### 4.3 SDK Compile
```shell
cd ~/workdir
./build.sh  # build full package
./build.sh kernel # build kenerl
./build.sh uboot  # build uboot

filesystem
./output/update/Image/rootfs.img -> ../../../debian/linaro-rootfs.img
```

## 8. Application Layer Development

### 8.1 Led->red/green/blue

This board has six leds can be controlled by software


- led on

```shell
  echo 1  > /sys/class/leds/ledaux/brightness
  echo 1  > /sys/class/leds/led1/brightness
  echo 1  > /sys/class/leds/led2/brightness
  echo 1  > /sys/class/leds/ledwifi/brightness
  echo 1  > /sys/class/leds/led4g/brightness
  echo 1  > /sys/class/leds/ledzgw/brightness
```

- led off

```shell
  echo 0  > /sys/class/leds/ledaux/brightness
  echo 0  > /sys/class/leds/led1/brightness
  echo 0  > /sys/class/leds/led2/brightness
  echo 0  > /sys/class/leds/ledwifi/brightness
  echo 0  > /sys/class/leds/led4g/brightness
  echo 0  > /sys/class/leds/ledzgw/brightness
```

### 8.2 Button
This board has one button can used by software, when pressed, the system will auto call the button script in the `/etc/rc.button/battery_power_switch`

- button value

```shell
cat /sys/class/gpio/gpio85/value
```
